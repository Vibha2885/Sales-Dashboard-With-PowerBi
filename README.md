# 📊 Sales Dashboard (Power BI)

An interactive **Power BI dashboard** designed to analyze sales data and generate actionable business insights. This project focuses on identifying trends, monitoring KPIs, and supporting data-driven decision-making.

---

## 🚀 Features

* 📈 Sales and revenue trend analysis
* 🌍 Region-wise performance breakdown
* 🧮 KPI tracking (Total Sales, Profit, Growth)
* 🎛️ Interactive filters and slicers
* 📊 Dynamic visualizations for better insights

---

## 🛠️ Tools & Technologies

* **Power BI**
* **Microsoft Excel / CSV Dataset**
* **Data Cleaning & Transformation (Power Query)**

---

## 📊 Project Highlights

* Analyzed **10,000+ sales records** to identify key business trends
* Built interactive dashboard improving data visibility and reporting
* Identified top-performing regions and products
* Simplified complex data into **easy-to-understand visuals**
